## Game: SOS Sederhana

Saya menambahkan game SOS berbasis web sederhana di repositori ini. Untuk mencoba secara lokal:

- Buka file [index.html](index.html) di browser Anda.
- Atau jalankan server lokal sederhana dari direktori proyek, misalnya:

```bash
# menggunakan Python 3
python3 -m http.server 8000
# lalu buka http://localhost:8000/index.html
```

Aturan singkat:
- Pilih huruf `S` atau `O`, lalu klik kotak kosong pada papan 3x3.
- Susun urutan "SOS" untuk mendapat poin (horizontal, vertikal, diagonal).
- Jika mendapat poin, pemain yang sama bermain lagi; jika tidak, giliran berganti.
